# Backyard Chickens Affiliate Site

Beginner-friendly backyard chicken keeping guide with Amazon affiliate monetization.

## Stack
- Next.js 15 + Tailwind CSS
- Markdown content in `src/content/articles/`
- Amazon affiliate tag: `backyardchx-20` (TBD - need to create)
- Deployed on Vercel via git push

## Content Strategy
- Breed guides (50+ breeds, each its own article)
- Gear reviews & comparisons (coops, feeders, waterers, heat lamps, etc.)
- Care guides (seasonal, beginner, troubleshooting)
- "Best X for Y" buyer's guides (highest conversion intent)

## Research Sources
- Reddit: r/BackYardChickens, r/chickens, r/homestead
- University agricultural extension sites (free, authoritative)
- Amazon reviews for product content
- USDA poultry resources
